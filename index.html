What is social media?

A social media platform is an online digital service or website that allows people to share and create content. For example, Facebook and Instagram are social media platforms because they allow users to communicate and connect with each other online. It is a public place for people to interact with each other through posts and messages.

Does it include a program?

Yes, a social media platform does include a program. In fact, they include code from various coding languages, such as HTML, Python, JavaScript, and more! This code handles the user interface, data management of the user, communication, security, and performance. It is used to store information in databases, while also keeping information safe and secure. Overall, the code is extremely important, and there are numerous programs that help make social media efficient and effective.

Is it easy to tell what the code does?

Social media is an international platform that connects its users using an algorithm and suggests other connections based on specific factors. Social media is a nonphysical concept as it refers to a variety of technologies which facilitate the sharing of information and ideas amongst users throughout the platform and the internet. These platforms initiate interactions without needing a physical presence, therefore proving social media to be a nonphysical concept.

Even though social media platforms are non physical, they hold a massive online presence as over 5 billion people use it.
